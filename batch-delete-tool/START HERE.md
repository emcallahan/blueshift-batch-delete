## Quick Guide

Once the tool opens in your browser:

1. **Enter your Blueshift User API Key**
   - Get it from: Blueshift ‚Üí Account Settings ‚Üí API Keys
   - It's never saved - enter it each time

2. **Upload your CSV file**
   - Drag and drop or click to browse
   - See `customers.csv.example` for format

3. **Select the identifier column**
   - Choose which column has emails/IDs

4. **Click "Start Batch Delete"**
   - Watch real-time progress
   - Download logs and failed records

5. **Stop the tool when done**
   - Double-click `STOP - Blueshift Tool.command`
